Projecten

Stage 2022-2023 MaxServ

Vanaf 29 augustus tot 3 februari heb ik bij MaxServ stage gelopen. In deze tijd heb ik 3 projecten gedaan.

1. Lorem

In het eerste project moest ik een website genaamd Lorem maken. De website was bedoeld voor een denkbeeldig keukenbedrijf, met de website zou je voorbeelden van de keukens van het bedrijf kunnen zien. Ook zit er de mogelijkheid in om een mail te sturen vanuit de browser naar het bedrijf en kun je de tekst op de pagina vertalen.

2. Todo4u

In het tweede project moest ik een website genaamd Todo4u maken. Het doel van de applicatie moet een persoonlijke, beveiligde todo applicatie zijn die ook met andere groepen gedeeld moet kunnen worden als de gebruiker daar behoefte aan heeft.

3. MaxServ Routeplanner

Bij het derde project moesten Tibo en ik een website genaamd MaxServ Routeplanner maken. De website was bedoeld voor de medewerkers van MaxServ die veel moesten reizen, met de website zouden de medewerkers kunnen aangeven waar ze heen moeten en dan zou de website alle mogelijke routes tonen. Ook zit er de mogelijkheid in om deze route te kunnen delen met een andere collega en de route in je agenda zetten.
Hierbij moet gebruik gemaakt worden van Laravel framework voor de back-end en React voor de front-end.
